# Necry's Hackpad
A custom 9-key macropad built for Hack Club Blueprint.

## Features
- 9 keys in a 3x3 matrix
- SK6812MINI RGB LEDs (one per key)
- 0.91" OLED display with custom ASCII art reactions
- EC11 rotary encoder for volume control
- 2 layers: Apps launcher and Media/Productivity
- KMK firmware on XIAO RP2040
- Wireless-ready (planned upgrade to nRF52840)

## Layers
- Layer 0: App launcher (File Explorer, Spotify, Opera, Steam, itch.io)
- Layer 1: Media and productivity shortcuts

## OLED reactions
Custom ASCII art for every key action!
